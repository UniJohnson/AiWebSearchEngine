# AiWebSearchEngine

create venv using:
python3 -m venv myenv

to activate venv use:
```source myenv/bin/activate```

to install requirements use:
```pip install -r requirements.txt```

test the crawler at:
https://vm009.rz.uos.de/crawl/

start webserver with:
flask --app myapp.py run

run crawler with:
python3 crawler.py
